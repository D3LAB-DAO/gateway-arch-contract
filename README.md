## Getting Started

### Prerequisites

- Rust programming language
- Archway CLI

### Building

To build the project, run:

```sh
$ archway contracts build
```

### Deployment

To deploy the contract, configure your Archway CLI and use the following command:

```sh
$ archway contracts store gateway
```
